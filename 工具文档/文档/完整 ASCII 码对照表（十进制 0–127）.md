## 核心规律补充

1. **数字字符**：十进制 `48` 对应 `0`，后续数字依次递增 1，即 `数字 n 的 ASCII 值 = 48 + n`。

2. **大写字母**：十进制 `65` 对应 `A`，后续字母依次递增 1，即 `大写字母第 n 位的 ASCII 值 = 65 + n-1`。

3. **小写字母**：十进制 `97` 对应 `a`，**小写字母 ASCII 值 = 对应大写字母值 + 32**，这是大小写转换的核心依据。

4. **控制字符**：0–31 和 127 为不可打印字符，主要用于设备控制，编程中常用的有 `9（TAB）`、`10（换行）`、`13（回车）`、`32（空格）`。
